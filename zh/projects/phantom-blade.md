# 影之刃零：武侠朋克 (Kung Fu Punk)

## 1. 绝境下的复仇：66天残命
主角“魂”被诬陷为杀害首领的凶手，在重伤濒死之际被神秘名医救治，但生命仅剩 66 天。玩家必须在倒计时结束前，在充满暗影的“组织”中杀出血路。

## 2. 核心战力：重构杀伐之美
由动作导演 **谷垣健治**（《浪客剑心》真人版动作指导）操刀，游戏追求极致的冷兵器碰撞感。
* **瞬杀与格挡**：快节奏的攻防转换，每一招都充满电影剪辑般的流畅感。
* **影境美学**：融合了蒸汽朋克与传统武侠的“武侠朋克”视觉，展现荒凉而肃杀的江湖。

## 3. 正式定档：2026年9月9日全球同步发售
《影之刃零》（Phantom Blade Zero）已于北京时间2025年12月13日的 **The Game Awards（TGA）颁奖典礼** 上正式公布发售日期，定档于 **2026年9月9日** 全球同步发售。

该游戏将登陆 **PlayStation 5** 及 **PC**（Steam、Epic Games Store）平台，采用 **虚幻引擎5（Unreal Engine 5）** 打造，是一款**买断制单机动作RPG**。从最新预告片中可见，游戏在保留极致武侠动作的同时，进一步强化了阴谋与背叛交织的叙事氛围，玩家将在蒸汽朋克风的荒凉江湖中，挥舞双武器系统，面对动态反击的BOSS战，直至揭开笼罩整个“组织”的惊天秘密。

倒计时已开启——距离“魂”的66天残命终点，还有不到半年。准备好在9月9日，与命运一战吗？
