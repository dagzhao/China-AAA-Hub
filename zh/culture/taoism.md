# 道教 101：战斗背后的形而上学

在中国 3A 游戏中，战斗不只是肌肉的碰撞，更是对宇宙规律的掌握。以下是**道教**如何驱动你的手柄：

---

### ☯️ 阴阳：攻防转换的艺术
西方往往看重“正邪对立”，而道教讲究**“平衡”**。
* **游戏机制：** “阴”代表柔（闪避、招架、静止），“阳”代表刚（进攻、火焰、动作）。
* **战斗心得：** 当 Boss 发动狂暴的“阳”属攻击时，聪明的玩家会使用“阴”派技巧——不与其硬碰硬，而是“以柔克刚”，借力打力。

### ⚡ 五行：不只是元素伤害
五行系统是一套复杂的“相生相克”循环。

| 元素 | 相互作用 | 游戏逻辑 |
| :--- | :--- | :--- |
| **金** | 克木 | 锋利度与暴击伤害 |
| **木** | 克土 | 持续回复与控制 |
| **水** | 克火 | 冰冻与流体身法 |
| **火** | 克金 | 灼烧伤害与爆发 |
| **土** | 克水 | 强韧度与防御姿态 |

### 📜 符印：诸神的逮捕令
为什么英雄要对强大的妖魔扔“纸片”？
* **真相：** 那不是普通的纸，那是**“天条”**。在道教文化中，施法者如同宇宙法官。
* **游戏内：** 符咒本质上是对现实的“强制修改”。它以更高阶神明的名义，命令灵体停止活动、燃烧或消散。
